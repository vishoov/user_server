Welcome to the User Management API

This API allows you to manage users efficiently. Below are the available endpoints:

API Endpoints

  • GET /users - Retrieve a list of all users.
  • GET /users/{id} - Retrieve details of a specific user by ID.
  • POST /users - Create a new user.
  • PUT /users/{id} - Update an existing user by ID.
  • DELETE /users/{id} - Delete a user by ID.

How to Use

Use your preferred API client (e.g., Postman, cURL) to interact with the endpoints. Ensure you include the required headers and payloads as specified in the API documentation.
